#  LearnMate AI

## Overview

LearnMate AI is an AI-powered educational assistant that helps students learn effectively from YouTube video lectures. The system extracts video transcripts and uses Artificial Intelligence to generate summaries, study notes, quizzes, flashcards, viva questions, and important timestamps. Students can also interact with an AI chatbot to ask questions based on the video content.

## Features

* 🎥 YouTube Transcript Extraction
* 📄 AI-Powered Video Summarization
* 📝 Smart Note Generation
* 💬 AI Doubt Solver Chatbot
* ❓ Quiz Generation
* 🎤 Viva Question Generator
* 🃏 Flashcard Generation
* ⏱ Important Timestamp Extraction
* 🌐 Multilingual Notes Generation
* 📊 Learning Progress Tracker

## Tech Stack

* Python
* Gradio
* Groq / Llama
* LangChain
* FAISS
* Sentence Transformers
* YouTube Transcript API

## Installation

1. Clone the repository

```bash
git clone https://github.com/nehasusan06/innovex.git
cd innovex
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the application

```bash
python app.py
```
